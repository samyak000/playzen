# 🎬 PLAYZEN – Movie Website

PLAYZEN is a modern movie website built using pure frontend technologies. It showcases a collection of movies with a clean UI and smooth user experience — all without using any external APIs.

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  

---

## ✨ Features

- 🎥 Browse a curated collection of movies  
- 🖼️ Display movie posters and basic details  
- 🎨 Clean and modern user interface  
- 📱 Fully responsive design  
- ⚡ Fast and lightweight (no API calls)
